# 七、列表

## 1.无序列表

```undefined
- 我是一个列表
+ 我是一个列表
* 我是一个列表

注意：- + * 跟内容之间都要有一个空格
```

- 我是一个列表
+ 我是一个列表

* 我是一个列表

## 2.有序列表

```undefined
1. 列表内容
2. 列表内容
3. 列表内容

注意：序号跟内容之间要有空格
```

1. 列表内容
2. 列表内容
3. 列表内容

## 3.列表嵌套

```
1. 第一项：
    - 第一项嵌套的第一个元素
    - 第一项嵌套的第二个元素
2. 第二项：
    - 第二项嵌套的第一个元素
    - 第二项嵌套的第二个元素
```

1. 第一项：
    - 第一项嵌套的第一个元素
    - 第一项嵌套的第二个元素
2. 第二项：
    - 第二项嵌套的第一个元素
    - 第二项嵌套的第二个元素
